# 🚀 Multi-Client Service Platform

A role-based web application that allows **Admin, Merchant, and Customer** to interact within a single platform. This project simulates a real-world service marketplace using **HTML, CSS, and JavaScript**.

---

## 📌 Project Overview

The Multi-Client Service Platform is designed to demonstrate how multiple users can interact with a system based on their roles:

- 👑 **Admin** → Monitor platform activity
- 🧑‍💼 **Merchant** → Add and manage services
- 👤 **Customer** → Browse and book services

The project uses **localStorage** to simulate backend data handling.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser LocalStorage  

---

## 🎯 Features

### 🔐 Authentication
- Login & Signup system
- Role-based redirection
- Session tracking using localStorage

---

### 👑 Admin Features
- View all registered users
- View system statistics:
  - Total Users
  - Total Services
  - Total Bookings

---

### 🧑‍💼 Merchant Features
- Add new services
- Delete services
- View only their services
- View bookings for their services

---

### 👤 Customer Features
- Browse available services
- Book services
- Cancel bookings
- View only their bookings
